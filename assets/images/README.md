## Image Format

- Use **PNG** format for all sprites (supports transparency)
- Keep file sizes small for web performance
- Use transparent backgrounds for all sprites except background
- Recommended: Use 32-bit color depth for best quality

## Tools for Creating Sprites

- **Aseprite** - Pixel art editor (paid)
- **Piskel** - Free online pixel art tool
- **GIMP** - Free image editor
- **Photoshop** - Professional image editor
- **Krita** - Free digital painting tool

## Tips

1. Keep sprites simple and recognizable at small sizes
2. Use high contrast colors for visibility
3. Add glow effects for space aesthetic
4. Test sprites in-game to ensure they're visible
5. Maintain consistent art style across all assets
